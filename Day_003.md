# Day 3: Secure Root SSH Access

Your task is to disable direct SSH root login on all app servers within the Stratos Datacenter.

## Solution
1. SSH to app server 1 and modify the settings on /etc/ssh/sshd_config
```sh
ssh tony@173.21.3.20
sudo vi /etc/ssh/sshd_config
```
> change to "PermitRootLogin no"

2. Restart sshd service
```sh
sudo systemctl restart sshd
```

3. Verify SSH root login wth below command
```sh
ssh root@<app-server-ip> 
```

4. Repeat the above procedure on all the app servers.
